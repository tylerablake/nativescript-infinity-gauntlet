Time

NativeScript doesn't use HTML or the DOM, it uses it's own XML-esque markup

In NativeScript everything must be inside of a container


There are 6 layout containers

  • Absolute (define the x,y position for the top left of the layout)
  • Dock (force the elements to dock to the top, bottom, left, or right of the layout)
  • Grid (similar to Bootstrap but you get to define your column and row sizes, etc)
  • Stack (stacks elements on top of eachother or beside each other)
  • Wrap (similar to StackLayout but wraps instead of stacking)
  • Flexbox (a non-conforming implementation of the CSS Flexible Box Layout)

Now ScrollView is NOT a layout container but can be treated like one
